# Image placement guide

Place these image files directly in `public/images/` so the site displays correctly.

## Required files (exact filenames)

| File | Used for |
|------|----------|
| `logo.png` | Navbar and footer |
| `butter-chicken.jpg` | Featured item card, gallery |
| `shawarma-loaded-fries.jpg` | Featured item card, gallery |
| `philly-cheesesteak-burger.jpg` | Featured item card, gallery |
| `tacos.jpg` | Featured item card, gallery |
| `mezze-platter.jpg` | Hero (home), about section, menu banner, gallery |

## Folder structure

```
public/images/
├── logo.png
├── butter-chicken.jpg
├── shawarma-loaded-fries.jpg
├── philly-cheesesteak-burger.jpg
├── tacos.jpg
└── mezze-platter.jpg
```

## Optional

- **`og/og.jpg`** — 1200×630 for social sharing (metadata in layout).
